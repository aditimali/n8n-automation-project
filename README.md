 🍽️ AI Restaurant Assistant

An AI-powered Restaurant Assistant built using n8n, Google Gemini, Google Sheets, and WhatsApp integration. It answers customer queries, checks inventory, retrieves FAQs, accepts orders, and stores them automatically.

 Features
- AI Chatbot
- WhatsApp Integration
- FAQ Support
- Inventory Check
- Order Management
- Google Sheets Database
- Conversation Memory

 Tech Stack
- n8n
- Google Gemini
- Google Sheets
- WhatsApp API
